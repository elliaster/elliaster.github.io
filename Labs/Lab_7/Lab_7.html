What is a Favicon exactly?

A favicon is the icon on the tab of the web browser when looking at a website. Favicon, short for Favorite Icon, is usually found in several file formats, such as .ico, .png, or .gif. There are other formats that can be used, but many popular browsers don't support it.

How are they created?

Favicons are created in programs such as Adobe Photoshop, gimp, Paint, etc.

How did we create Favicons?

We created Favicons using the Color Pixelation Widget on Code.org
